# 🛡️ AI Email Spam Detection System

An AI-powered Email Spam Detection System built using **Machine Learning**, **TF-IDF**, and **Streamlit**. It classifies emails as **Spam** or **Not Spam** with a modern web interface.

## 🚀 Features

- 📧 Spam & Ham Email Detection
- 🤖 Machine Learning (LinearSVC)
- 🔤 TF-IDF Text Vectorization
- 📊 Confidence Score
- 📁 Batch CSV Prediction
- ☁️ Word Cloud
- 📈 Model Performance Dashboard
- 🎨 Interactive Streamlit UI

## 🛠️ Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Joblib
- WordCloud
